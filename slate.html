Slate: Slide Making in Web-Age

To run the demo :

  • Go to Github repository
  • Copy the code in the file named myPres.slate which is located in Demo folder.
  • Save the code in a file with a name like : demo.slate, using any word-editing software like notepad.
  • Browse to the saved file and upload it.
  • Presentation will be generated which can be navigated using arrow keys (Enter for animation).
  • ReadMe.md (Github) will give further instructions.
  • Documentation.pdf (Github) explains the tool in details.
